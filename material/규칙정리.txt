Q-marble 규칙 설명

1. 주사위를 던져서 나온 수(1~3) 만큼 칸을 이동한다.
2. 각 칸에서 퀴즈를 풀어야 한다.
	-나라 퀴즈 칸: 12개 
	-프로그래밍 퀴즈 칸: 3개
3. 나라 퀴즈 칸에 도착하면 상중하 난이도를 선택한다.
	- 문제를 맞췄을 시
	상 : +20000  
	중 : +10000
	하 : +5000
	- 문제를 틀렸을 시
	상 : -20000  
	중 : -10000
	하 : -5000
4. 프로그래밍 퀴즈 칸에 도착하면 프로그래밍 문제 12개 중 랜덤으로 문제가 주어진다.
	- 문제를 맞췄을 시
	+30000
	- 문제를 틀렸을 시
5. 시작 칸에 도착할 시 +20000이 부여된다.
6.  게임 엔딩 조건
	-한 명이 파산하거나 한 명이 먼저 250,000을 넘기면 승리하게 된다.
